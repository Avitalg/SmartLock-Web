About

This system was created as part of a final project of the Department of Software Engineering at Shenkar College of Engineering and Design. The project was created in order to develop and implement the methods and tools that were acquired during the four years of study in software engineering at Shenkar.

The idea for the project was created as a result of the personal need we encountered. We have taken upon ourselves to solve this need by using the technologies we have learned.

The project was built by a collaboration between 3 partners, each of whom developed another part of the project.

The project was carefully planned for a number of months in which the development language, the development method, the system structure and the database structure were thought of.

Of course we give credit for the pictures on the site that were taken from the site https://pixabay.com.
